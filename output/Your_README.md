## Description
A command line project that incorporates the use of node.js and various functions to allow the user to create a readme for any project they desire.

## Installation
This requires the use of node.js. Please make sure this is installed before use.

## Usage
clone this repository and make sure that inquirer is also installed within the project. Then the user will be able to follow these prompts to populate a markdown file, generated after the program is completed.
